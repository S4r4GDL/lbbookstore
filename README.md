# LBbookstore (en-us)

### Sara Gonçalves de Lima

This is my first project for the Web Programming 1 course!

**Chosen Entity:** Book

**Main Technologies:**

-   Java 21
-   Spring Boot
-   PostgreSQL

**Essential Features:**

-   Book Management: CRUD operations
-   Search by Attribute: Operations to search for specific books by attributes such as title, author, etc.

**Structure and Dependencies:** Spring Web, Spring Data JPA, Lombok, and MapStruct
